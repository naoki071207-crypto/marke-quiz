# マーケ用語クイズ

マーケティング用語(333語)を、一覧・フラッシュカード・4択クイズで学べるWebアプリ。

## 使い方

ブラウザで `index.html` を開くだけ。

ローカルサーバーで開く場合:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | UI構造 (3モードのタブ切替) |
| `style.css` | スタイル (グラデ・カードめくりアニメ・レスポンシブ) |
| `data.js` | 用語データ (333語、7カテゴリ) |
| `app.js` | アプリロジック (検索・カード・クイズ) |

## 3つの学習モード

- **📚 一覧**: 全用語をカード表示。検索＋カテゴリ絞込
- **🃏 フラッシュカード**: タップでめくる暗記カード。シャッフル対応、←→キーで前後移動
- **📝 4択クイズ**: 出題数・カテゴリ・出題方向を設定可。結果画面で間違えた用語リスト表示

## 用語の追加・編集

`data.js` の `TERMS` 配列に下記形式で追加:

```js
{ id: 999, term: "用語名", full: "正式名称", category: "指標",
  description: "定義...\n\n例: ...\n\nポイント: ..." },
```

`description` 内の `\n\n` で段落を分けると、UIに反映されます。
